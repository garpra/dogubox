# Dogubox

**Dogubox** is a personal Arch Linux setup toolkit focused on automation.

## Installation

```bash
git clone https://github.com/garpra/dogubox.git
cd dogubox
```

## Usage

```bash
./dogubox
```

## Structure

```bash
dogubox/
├── dogubox                 # Main wrapper script
├── scripts/                # Supporting scripts
└── src/                    # Custom tools
```
